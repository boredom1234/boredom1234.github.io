Install

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://boredom1234.github.io/ (Give it the name "boredom1234 Repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "boredom1234 Repository", and install repository.boredom1234-2.0.1.zip.
  • Restart Kodi
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "boredom1234 Repository"
  • Go into the "Video add-ons" section in the repo, and you'll find all video add-ons
  • Install JioTV from it
  • Configure it as needed (Login and other settings as per your need)
  • Now go to Addons, then install from repository and then PVR Clients. Select Simple IPTV Client and install it
  • Setup the JioTV addon using configure and you are done.
